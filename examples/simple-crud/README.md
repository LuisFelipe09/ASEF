# Ejemplo 1: Microservicio CRUD
Este directorio contiene una implementación de referencia de un microservicio simple siguiendo el framework ASEF.

## Estructura
* `docs/adr/`: Decisiones de arquitectura.
* `features/`: Tests de comportamiento (Gherkin).
* `docs/diagrams/`: Diagramas visuales.
* `src/`: Código fuente (placeholder).

## Cómo leer esto
Empieza por el ADR (`docs/adr/001-microservicio-usuarios.md`) para entender el "por qué", luego revisa los tests Gherkin para entender el "qué", y finalmente el diagrama para ver el "cómo".
